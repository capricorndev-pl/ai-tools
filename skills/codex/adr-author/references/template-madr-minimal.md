# MADR (minimal) template

Use when the decision is small, the winner is obvious, or the user asks for something "quick".

```markdown
# {short imperative title}

- Status: {Proposed | Accepted}
- Date: {YYYY-MM-DD}

## Context and Problem Statement

{1–3 sentences. What forced the decision.}

## Considered Options

- Option A
- Option B

## Decision Outcome

Chosen option: **{option}**, because {one sentence}.

### Consequences

- **Positive:** {…}
- **Negative:** {…}
```

If you find yourself wanting more structure than this, switch to the full MADR template.
